# Crypto Whale Tracker

A simple tool that tracks large blockchain transactions and monitors whale wallet activity.

## Features
- Track large blockchain transactions
- Monitor whale wallet activity
- Detect major crypto transfers
- Educational tool for understanding on-chain analytics

## Technologies
- Python
- Blockchain APIs
- Web3 data integration
- REST API

## Purpose
This project was created to explore on-chain data and understand how large wallet transactions and whale activity can be tracked using blockchain analytics tools.

## Future Improvements
- Real-time whale transaction alerts
- Support for multiple blockchain networks
- Wallet activity dashboard
- Integration with market sentiment analysis
